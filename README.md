## 🎯 Objectif

Ce dépôt contient l'exploration et l'analyse des annonces immobilières pour le département du Haut‑Rhin (68). L'objectif est de :
- Examiner l'évolution du prix au m² dans le temps,
- Réaliser des tests statistiques et des analyses par commune,
- Produire des visualisations .
